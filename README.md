## Requirements

- Swift 5 and above
- iOS 12 and above

## Installation

```
pod 'SATrackingSDK'
```

or

```
pod 'SATrackingSDK', :git => 'https://github.com/solutionarchitectstech/ios_tracker_sdk_release.git', :tag => '1.1.7'
```
